# Chapter 5: Langflow AI Agents

This chapter contains Langflow-based AI agent workflows and related applications.

## Contents

### Playwright Report Analyzer 2
- **Location**: `Playwright Report Analyzer 2/`
- **Description**: Full React Vite application that compares two Playwright test report JSON files and surfaces flaky tests, consistent failures, and rerun recommendations using Langflow + Groq.
- **Prerequisites**: Node.js 18+, Langflow running at `http://localhost:7860`, matching flow ID and API key
- **Build**: `npm install && npm run dev`
- **Port**: `:5173`
- **Details**: See `Playwright Report Analyzer 2/README.md` for full setup and usage

### Flakey Test Cases
- **Location**: `Flakey test cases/`
- **Description**: Sample flaky test execution reports and JSON artifacts for analysis and demonstration
